# logistica-python
Scripts desenvolvidos durante a Pós em Engenharia de Software na Descomplica
Este script em Python foi desenvolvido para resolver um problema comum em setores de logística e emissão fiscal: o acúmulo de arquivos desorganizados (CT-es, Notas Fiscais e planilhas de rastreio).

O que ele faz:

    Identifica automaticamente a extensão dos arquivos (.pdf, .xlsx, .txt).

    Cria pastas específicas para cada categoria.

    Move os documentos para seus respectivos locais, otimizando o tempo de busca e organização.

🇺🇸 English

This Python script was developed to solve a common challenge in logistics and tax departments: the accumulation of disorganized files (Tax documents and tracking spreadsheets).

Main features:

    Automatically identifies file extensions (.pdf, .xlsx, .txt).

    Creates specific folders for each category.

    Moves documents to their respective locations, optimizing search time and workflow.

🛠️ Tecnologias Utilizadas (Tech Stack)

    Python 3.x

    Bibliotecas: os, shutil (Built-in)

    


    ## 📊 Banco de Dados (SQL)
Além do Python, incluí um projeto de **SQL** que simula um banco de dados de inventário.
